Guía Rápida de Etiquetas HTML

Una referencia esencial para desarrolladores web

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Esta guía cubre las etiquetas más importantes y útiles.

Estructura Básica

<!DOCTYPE html>

Define que el documento es HTML5

<!DOCTYPE html>

<html>

El contenedor raíz de una página HTML

<html lang="es"> <!-- contenido de la página --> </html>

<head>

Contiene metadatos/información sobre el documento

<head> <title>Título de la página</title> <meta charset="UTF-8"> </head>

<body>

Contiene el contenido visible de la página

<body> <h1>Mi página web</h1> <p>Contenido de la página</p> </body>

Metadatos

<title>

Define el título del documento (mostrado en la pestaña del navegador)

<title>Mi Página Web</title>

<meta>

Provee metadatos sobre el documento HTML

<meta charset="UTF-8"> <meta name="description" content="Descripción de mi página"> <meta name="keywords" content="HTML, CSS, JavaScript"> <meta name="viewport" content="width=device-width, initial-scale=1.0">

<link>

Usado para enlazar recursos externos como hojas de estilo CSS

<link rel="stylesheet" href="styles.css">

<style>

Define estilos CSS internos

<style> body { font-family: Arial, sans-serif; } </style>

Encabezados y Texto

<h1> a <h6>

Encabezados o títulos (h1 es el más importante, h6 el menos)

<h1>Encabezado principal</h1> <h2>Subtítulo</h2> <h3>Apartado</h3>

<p>

Define un párrafo

<p>Este es un párrafo de texto.</p> <p>Este es otro párrafo.</p>

<br>

Inserta un salto de línea (etiqueta auto-cerrada)

Texto en una línea<br>Texto en la siguiente línea

<hr>

Define una línea horizontal (etiqueta auto-cerrada)

<p>Texto arriba</p> <hr> <p>Texto abajo</p>

<strong>

Texto importante (normalmente se muestra en negrita)

<strong>Texto importante</strong>

<em>

Texto enfatizado (normalmente se muestra en cursiva)

<em>Texto enfatizado</em>

<span>

Contenedor en línea para texto

<p>Texto <span style="color:red">rojo</span> en el párrafo</p>

Enlaces e Imágenes

<a>

Define un hipervínculo

<a href="https://www.ejemplo.com">Enlace a ejemplo.com</a> <a href="pagina.html" target="_blank">Abrir en nueva pestaña</a>

<img>

Inserta una imagen (etiqueta auto-cerrada)

<img src="imagen.jpg" alt="Descripción de la imagen" width="500">
Nota: El atributo alt es importante para accesibilidad y SEO.

Listas

<ul>

Lista desordenada (con viñetas)

<ul> <li>Elemento 1</li> <li>Elemento 2</li> </ul>

<ol>

Lista ordenada (con números)

<ol> <li>Primer elemento</li> <li>Segundo elemento</li> </ol>

<li>

Elemento de una lista

<li>Elemento de lista</li>

Formularios

<form>

Contenedor para elementos de formulario

<form action="/procesar.php" method="post"> <!-- elementos del formulario --> </form>

<input>

Campo de entrada (etiqueta auto-cerrada)

<input type="text" name="nombre" placeholder="Escribe tu nombre"> <input type="password" name="clave"> <input type="email" name="correo"> <input type="number" name="edad"> <input type="checkbox" name="opcion"> <input type="radio" name="genero"> <input type="submit" value="Enviar">

<textarea>

Área de texto multilínea

<textarea name="mensaje" rows="4" cols="50"> Texto por defecto </textarea>

<select>

Lista desplegable de opciones

<select name="color"> <option value="rojo">Rojo</option> <option value="verde">Verde</option> <option value="azul">Azul</option> </select>

<button>

Botón clickeable

<button type="button">Botón normal</button> <button type="submit">Enviar formulario</button> <button type="reset">Restablecer</button>

<label>

Etiqueta para un elemento de formulario

<label for="nombre">Nombre:</label> <input type="text" id="nombre" name="nombre">

Tablas

<table>

Define una tabla

<table> <tr> <th>Nombre</th> <th>Edad</th> </tr> <tr> <td>Juan</td> <td>25</td> </tr> </table>

<tr>

Fila de una tabla

<tr> <td>Celda 1</td> <td>Celda 2</td> </tr>

<td>

Celda de datos en una tabla

<td>Contenido de la celda</td>

<th>

Celda de encabezado en una tabla

<th>Encabezado de columna</th>

Elementos Semánticos (HTML5)

<header>

Encabezado de una sección o página

<header> <h1>Título del sitio</h1> <nav>...</nav> </header>

<nav>

Sección de navegación

<nav> <a href="/">Inicio</a> <a href="/nosotros">Nosotros</a> </nav>

<main>

Contenido principal de la página

<main> <!-- contenido principal --> </main>

<section>

Sección temática de contenido

<section> <h2>Título de sección</h2> <p>Contenido de la sección</p> </section>

<article>

Contenido auto-contenido e independiente

<article> <h2>Título del artículo</h2> <p>Contenido del artículo</p> </article>

<aside>

Contenido relacionado indirectamente con el contenido principal

<aside> <h3>Contenido relacionado</h3> <p>Información adicional</p> </aside>

<footer>

Pie de página de una sección o página

<footer> <p>Derechos reservados © 2023</p> </footer>

Notas importantes:

  • HTML5 introdujo elementos semánticos que mejoran la accesibilidad y el SEO
  • Las etiquetas deben cerrarse apropiadamente
  • Los atributos siempre deben ir entre comillas
  • Usa siempre el atributo alt en las imágenes para accesibilidad
